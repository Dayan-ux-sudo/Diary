# Diary Web App
This is a web-based platform diary that includes both the Frontend and a Backend.
##Frontend
The Frontend is written in React Javascript Language(tsx).
The Frontend has features like statistics that show the user the number of total tasks, completed tasks, todays tasks and completion rate.
One can be able to add a task and give the description of that task, its date pririty and actegory.
It also includes all your tasks.
Running the Frontend use #npm run dev#.
##Backend
It is written in React Javascript language.
It connects the Fronted to a Google Cloud Database known as the Firebase. 
When running the backend use #npm run dev#.
